## list

Retrieve a list of tags for the authenticated workspace.

### Example Usage

```typescript
import { Dub } from "dub";

async function run() {
  const sdk = new Dub({
    token: "<YOUR_BEARER_TOKEN_HERE>",
    workspaceId: "<value>",

  const result = await sdk.tags.list({});

  // Handle the result
  console.log(result)

run();
```
